## Сборник решений задач с платформы Leetcode.com

Локальное тестирование работает через jest. Для установки зависимостей необходимо ввести
```
 npm i
```

Для удобного создания папки с задачей используйте скрипт 
```
npm run gen string-compression
```
где string-compression - это Slug из URL задачи.
Скрипт автоматически создаст папку в tasks/[уровень сложности],
а затем добавит тесты (входные параметры), ожидаемый результат нужно ввести самому

### Важно
По контестам в воскресенье этот скрипт не работает. Но можно создать заготовки из шаблона через
```
npm run new НАЗВАНИЕ ЗАДАЧИ (можно с пробелами)
```

Создаст папку с файлами index.js и index.test.js в tasks. Оттуда ее перенести в нужную сложность
